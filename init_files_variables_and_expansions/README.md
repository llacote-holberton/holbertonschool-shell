This folder holds all scripts related to Shell, init files, variables and expansions.

| Filename                    | Action(s) summary                                                                                  |
| --------------------------- | -------------------------------------------------------------------------------------------------- |
| 0-alias                     | Creates the most malevolent alias EVER (who is evil enough to have proposed this???)               |
| 1-hello_you                 | Says hello to currently authenticated user.                                                        |
| 2-path                      | Appends a '/action' path to the list of paths in which shell searches for a desired program.       |
| 3-paths                     | Counts the number of directories in the PATH environment variable.                                 |
| 4-global_variables          | Lists (all)  environment variables.                                                                |
| 5-local_variables           | Shows all local variables and environment variables, and functions.                                |
| 6-create_local_variable     | Creates a new local variable.                                                                      |
| 7-create_global_variable    | Creates a new global variable.                                                                     |
| 8-true_knowledge            | Prints the addition of 128 and 1337 stored in TRUEKNOWLEDGE environment variable.                  |
| 9-divide_and_rule           | Dividing is conquering (displays result of a division of POWER by DIVIDE).                         |
|10-love_exponent_breath      | Shows off the power LOVE brings to our BREATH (displays the result of exponential combination).    |
|11-binary_to_decimal         | Converts binary number stored in BINARY enviromnment var in decimal.                               |
|12-combinations              | Expand the realm of combining two random characters, sorted, excluding specific 'oo' case.         |
|13-print_float               | Displays a number stored in environment variable NUM formatted with two decimal digits.            |
|14-decimal_to_hexadecimal    | Converts decimal number stored in DECIMAL enviromnment var in binary.                              |
