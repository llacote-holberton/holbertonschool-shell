This folder holds mini-scripts representing
  the script use of simple commands.

List of exercices and related filename
-  0-current_working_directory - Prints the absolute path name of the current working directory.
-  1-listit                    - Displays the contents list of your current directory.
-  2-bring_me_home             - Changes the working directory to the user's home directory.
-  3-listfiles                 - Displays current directory contents in a long format.
-  4-listmorefiles             - Displays current directory contents in long format including hidden files.
-  5-listfilesdigitonly        - Displays current directory contents with UID & GID instead of names.
-  6-firstdirectory            - Creates a 'my_first_directory' directory in /tmp.
-  7-movethatfile              - Moves a file 'betty' from /tmp into aforementioned directory.
-  8-firstdelete               - Deletes the moved file 'betty'
-  9-firstdirdeletion          - Deletes the previously created directory.
- 10-back                      - Changes working directory to the one user was into previously. 
- 11-lists                     - Lists ALL files in current + parent directory as well as /boot one.
- 12-file_type                 - Displays the "type" of a /tmp/iamafile file if exists.
- 13-symbolic_link             - Creates a symbolic link of 'ls' builtin as __ls__ in current directory.
- 14-copy_html                 - Copies "in update mode" all "html files" present in current dir to parent one.
- 15-lets_move                 - Moves all files in current dir starting with one uppercase to /tmp/u/.
- 16-clean_emacs               - Deletes all files of current dir which filenames end exactly with '~' character.
- 17-tree                      - Creates a "subtree" of directories ./welcome/to/school from current directory.
