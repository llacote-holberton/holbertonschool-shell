This folder holds all scripts related to Text manipulations and I/O Redirections trainings in Shell.

| Filename                    | Action(s) summary                                                                                  |
| --------------------------- | -------------------------------------------------------------------------------------------------- |
| 0-hello_world               | Displays "Hellow World" followed by empty line on standard output.                                                                  |
| 1-confused_smiley           | Displays a "confused smiley" in ASCII art.                                                         |
| 2-hellofile                 | Gets the classic "list all users's essential informations in a (VERY) counter-intuitive way".      |
| 3-twofiles                  | Shows contents of both passwd and hosts essential system files.                                    |
| 4-lastlines                 | Displays the last  10 lines of /etc/passwd.                                                        |
| 5-firstlines                | Displays the first 10 lines of /etc/passwd.                                                        |
| 6-third_line                | Shows only 3rd line of 'iacta' file without using sed (I guess it's up to grep then?).             |
| 7-file                      | Creates a hellish abomination of filename made of special characters, which should never exist.    |
| 8-cwd_state                 | Shows a classic case of "snapshotting" dir contents into a text file for later analysis. :)        |
| 9-duplicate_last_line       | Duplicates the last line of 'iacta' file because some people weren't attentive enough xd.          |
|10-no_more_js                | Erase javascript files from existence downwards from current point, "Thanos style" MWAAAAHAHAHAHA. |
|11-directories               | Counts all directories downwards from current dir because more fun than counting sheeps.           |
|12-newest_file               | Lists the ten most recently touched (created/modified) files in current dir, useful in many cases. |
|13-unique                    | Processes a text input to list out all words appearing only once.                                  |
|14-findthatword              | Gets lines from passwd file which have root(s) inside (apparently case-insensitve).                |
|15-countthatword             | Displays number of lines with 'bin' in passwd file (to be or to have bin that is the question ;)). |
|16-whatsnext                 | Shows lines containing root with their following 3 lines (roots gotta spread?).                    |  
|17-hidethisword              | Shows all the lines which do NOT have 'bin' because bins are always dirty and we don't like that.  |
|18-letteronly                | Displays only lines of sshd config which start with a letter (because we are wo-men of letters ;)).| 
|19-AZ                        | Replaces characters from input (A -> Z, c -> e) to create a new, weird, impractical language.      |
|20-hiago                     | Removes all kind of c from input to create custom Chaos xd.                                        | 
|21-reverse                   | Reverses input's characters orders to help people learn "Verlan", a popular French slang.          | 
|22-users_and_homes           | Uses passwd efficiently now to get ordered list of users with their homes (no doxxing though).     |
